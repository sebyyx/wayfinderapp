# Wayfinder Web

Marketing site (`wayfinderapp.life`) + admin dashboard (`admin.wayfinderapp.life`)
for the Wayfinder iOS app.

```bash
cp .env.example .env   # fill in Supabase keys
npm install
npm run dev            # http://localhost:3000
```
